O que é o HTML

HTML significa HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto. O HTML é um dos componentes básicos da web, com ele adicionamos conteúdos para criar uma estrutura básica para o site. Essa estrutura serve para exibir os textos e elementos como; imagens, vídeos, documentos, formulários e etc... E para adicionar essas informações utilizamos as TAGS para dar formas as informações inseridas no site.

Para que essa forma tenha um design desejado, é preciso a utilização do CSS e JavaScript. Para simplificar melhor essa informação. Imagine que o HTML é o esqueleto e o CSS e JavaScript é o restante do corpo, cabeça, braços, torso, pernas, mãos e pés. Permitindo assim criar um corpo de acordo com o estilo que você desejar.

Qual a sua função?

O HTML serve para criar o esqueleto do site. Assim como este site, que foi iniciado do zero, foi necessário a utilização do HTML para criar a sua estrutura inicial. E para dar forma ao site é necessário acrescentar as TAGS, com ela podemos definir o que em um determinado momento do site passa a ser um título, ou um campo para o preenchimento de um formulário, até mesmo para adicionar vídeos ou imagens como mencionado no texto acima.
A seguir falaremos sobre as tags, para melhor orienta-lo sobre o assunto.



O que são Tags?

As Tags são instruções do HTML para informar o navegador como interagir com determinado texto, imagem, documento ou formulário no navegador.

Qual a sua função no HTML?

As Tags trabalham de forma invisível ao olho do úsuário enquanto navega pelo site, porém elas são ferramentas chave para instruir o navegador de como se comportar diante o usuário.

As tags mais utilizadas no HTML.

  • Vamos iniciar falando das Tags h1 ao h6, talvez a Tag mais conhecida entre todas as outras Tags.
    Elas servem para definir títulos, e sua numeração serve para determinar o grau de importância de cada um deles, começando do h1 até o h6. É indicado apenas utilizar o h1 uma única vez. É importante lembrar também de seguir sempre a ordem, evitar pular uma numeração para que os robos do Google consigam efetuar uma boa leitura e pontunar o seu SEO para um bom ranqueamento no Google.

    • Exemplo:

    1. h1

    2. h2

    3. h3

    4. h4

    5. h5
    6. h6

  • Temos o p que serve para definir um parágrafo.

  • A utilização da Tag ol serve para criar uma lista ordenada, seja ela por números ou ordem alfabética. Essa alteração depende do que é feito na programação no CSS ou na adição do style no código HTML.

    2. Molhe as bolachas no leite e reserve.

    3. Em uma panela, leve todos os ingredientes ao fogo médio e misture até obter uma consistência grossa e cremosa.

      4. Creme de Chocolate:

    4. Repita o processo feito no creme branco.

      5. Para cobertura:

    5. Bata as claras em neve com o açúcar até obter um creme consistente, adicione o creme de leite e misture delicadamente.

    6. Em um refratário grande, despeje o creme branco, metade das bolachas, creme de chocolate, bolachas e claras em neve.

  • A Tag li é utilizada dentro das Tags ol e ul, para definir os itens, e dependendo do style, você consegue personalizar com simbolos, números ou letras. Quando você visualiza uma lista e ela se encontra ordenada por números, por exemplo: 1,2,3... Essas numerações dentro do HTML estão sendo representadas pelo li
